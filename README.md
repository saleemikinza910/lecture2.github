 ✨ My Story – Kinza Saleemi

A beautifully designed personal “My Story” page that showcases a software engineering student’s journey, hobbies, and creative side. The page features an elegant color palette from Coolors, three unique professional font combinations, and a responsive layout with subtle background textures.

🔗 Live preview: open index.html in any modern browser.

📖 About the project

This single‑page HTML/CSS/JS project tells my story (Kinza Saleemi) – a software engineering student who loves traveling, reading, learning languages, and watching meaningful series. It’s built to be:

Visually refined – using a curated color scheme: deep navy, burnt sienna, warm sand, and slate.
Typography‑rich – three hand‑picked font combos (Lora+Inter, Playfair+Roboto, Montserrat+Merriweather) that you can switch with one click.
Fully responsive – looks great on desktop, tablet, and mobile.
Personal & narrative – includes real details about coding, travel experiences, language learning (Urdu, English, Japanese), favorite series, and current reads.

🔤 Font combinations (experiment with 3 styles)

Three professional font pairs are available via the buttons at the top of the page:

Combo 1 (Editorial) – Lora (serif headings) + Inter (sans‑serif body)
Combo 2 (Classic) – Playfair Display (serif headings) + Roboto (sans‑serif body)
Combo 3 (Modern) – Montserrat (sans‑serif headings) + Merriweather (serif body)
Click any combo button to instantly change the page’s typography while preserving all spacing and rhythm.
